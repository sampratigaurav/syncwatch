## 2026-06-21 - Remove inline style tags in React components
**Learning:** Found multiple React components injecting global `<style>` blocks (e.g. keyframes) directly into the DOM during render. Because React doesn't natively scope raw `<style>` tags, they re-evaluate on component render and can cause unnecessary layout thrashing and DOM bloat (especially if the component mounts multiple times, or conditionally renders).
**Action:** Always extract non-scoped global styles and animations from inline `<style>` tags in React components and move them to global stylesheets (like `index.css`) to improve rendering performance and prevent style recalculation thrashing.
